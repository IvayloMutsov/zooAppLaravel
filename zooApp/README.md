# Zoo Shop Management System

Това е система за управление на зоомагазин, разработена с Laravel + SQLite.

## Функционалности

- CRUD за животни, породи и видове животни
- Административен панел с authentication
- Добавяне на потребители за админ панела
- Качване на изображения за животни
- Търсене по име, порода, вид животно, дата на раждане
- Публична част с преглед на последните добавени животни

## Технологии

- Laravel 10/11
- PHP 8+
- SQLite база данни
- Blade templates
- Bootstrap 5 за front-end
